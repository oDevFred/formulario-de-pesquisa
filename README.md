# Formulário de Pesquisa

Este é um projeto de formulário de pesquisa responsivo e acessível, desenvolvido como parte de um desafio de desenvolvimento web. O formulário foi criado utilizando HTML e CSS, seguindo boas práticas de design e usabilidade.

## Funcionalidades

- **Campos obrigatórios**: Nome, e-mail e idade.
- **Validações HTML5**:
  - E-mail formatado corretamente.
  - Campo de número aceita apenas valores numéricos.
  - Validação de intervalo para a idade (entre 12 e 120 anos).
- **Menu suspenso**: Permite selecionar como o usuário conheceu o serviço.
- **Radio buttons**: Para avaliar a probabilidade de recomendação.
- **Checkboxes**: Para selecionar os serviços utilizados.
- **Textarea**: Para comentários adicionais.
- **Botão de envio**: Estilizado e funcional.

## Tecnologias Utilizadas

- **HTML5**: Estrutura do formulário e validações nativas.
- **CSS3**: Estilização responsiva e design moderno.

## Como Usar

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/formulario-pesquisa.git
   ```
2. Abra o arquivo `index.html` no seu navegador.
3. Preencha o formulário e envie as informações.
